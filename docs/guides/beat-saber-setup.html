Setup Guide

Beat Saber on your Mac, in VR

From zero to slicing blocks — six steps, one app. Apple Silicon + any ALVR headset.

Before you start

  • Apple Silicon Mac running macOS 14 or newer
  • Headset with the ALVR client v20.14.0 installed (tested on Quest 3)
  • A Steam account that owns Beat Saber (v1.44.1 verified)
  • Optional, for USB streaming: brew install android-platform-tools
  • Optional, for game audio: brew install --cask blackhole-2ch && brew install switchaudio-osx
1

Download MoltenVR

Grab MoltenVR-Preview-vX.Y.Z.zip from the GitHub Releases page, unzip, and drag MoltenVR.app into /Applications. The preview isn't notarized yet, so on first launch right-click → Open. Everything works free for 7 days from first launch.

2

Set up your bottle

A bottle is the Windows environment your games live in. In the setup wizard (or the Bottles tab), type a name and hit Create — MoltenVR downloads and manages its own Wine, no Whisky or CrossOver needed. Then click Install DXMT to add the graphics layer. Full details in the companion Bottles & DXMT guide.

3

Install Steam

With your bottle selected, click Install Steam, then Launch Steam and log in to your account. First login can take a few minutes while Steam updates itself — let it finish.

4

Download Beat Saber

Inside Steam (running in your bottle), install Beat Saber from your library like on any PC. Version 1.44.1 is verified working. Prefer modded? Beat Saber 1.40.8 with BSIPA via BSManager is verified too — MoltenVR's game scan picks up BSManager multi-version installs automatically.

5

Register the runtime

Back in the Bottles tab, click Register OpenXR Runtime. This makes MoltenVR the bottle's OpenXR runtime, so the game's VR calls land in MoltenVR's frame capture instead of looking for SteamVR. The bottle row should now show all three checks: DXMT ✓  Runtime ✓  Steam ✓.

6

Launch the game

Press Start Stream, put your headset on, and open the ALVR app. On the Mac, open the Games tab, hit Scan Games, and click Patch next to Beat Saber if it shows stock dll — then Launch. Launching from the Games tab matters: it sets the Steamworks environment (SteamAppId) and handles BSIPA injection, so the game doesn't quit after 60 seconds.

The Games tab after a scan — Beat Saber detected on OpenXR, no patch needed, ready to Launch.
Tips. USB streaming (100 Mbps, no Wi-Fi jitter) needs android-platform-tools and the headset plugged in before you start the stream. Double-tap the menu button in VR to see your Mac desktop. macOS will ask for Screen Recording and Microphone permissions the first time those features are used — grant both.
Something flat or broken? If Beat Saber launches flat, re-check step 5 and the Patch state in the Games tab. If Steam misbehaves after a Wine update, the Bottles tab offers Update Wine (fixes Steam). Still stuck — join the Discord and open an issue with the game name.
